# YT Downloader
Simple youtube downloader using python with youtube-dl + ffmpeg

Installation
```
sudo apt update
sudo apt install ffmpeg
git clone https://github.com/prhnahmd/ytdl/
pip install -r requirements.txt
```

Usage
```
python3 ytdl.py
```
